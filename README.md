# TODO Tools

一个简单的命令行待办事项和任务管理工具。

## 功能特点

- 📝 待办事项管理
  - 添加、删除、编辑待办事项
  - 支持设置任务重要性和预计时长
  - 快速添加功能（默认45分钟，中等重要性）

- ⏰ 番茄钟功能
  - 实时进度显示
  - 系统通知提醒
  - 支持手动中断（Ctrl+C）
  - 任务完成后评分和总结

- 📊 任务记录
  - 记录任务执行情况
  - 支持满意度评分（1-5星）
  - 支持添加任务总结
  - 查看历史任务记录
